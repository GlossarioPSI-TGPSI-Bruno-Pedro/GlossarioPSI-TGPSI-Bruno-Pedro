Passing Parameters:
  It is the passage of variables from one method to another, such as the
main method for a method that can make a sum account. There are three types of passages:
Pass by value, Pass by name and Pass by reference.




Polymorphism:
  Allows abstract classes to represent the behavior of concrete classes
which they refer.




Programming language:
  It is a set of commands, easily memorable and with logical meanings that
when placed in a certain logical and sequential order can be read by a compiler.
