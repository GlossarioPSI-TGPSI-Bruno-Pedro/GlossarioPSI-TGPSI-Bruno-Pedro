Method:
  It is a set of code usually created before the main method that can be
any point in the program.




Method without return:
  It is a method that does not return any type of value after it is executed.




Method with return:
  It is a method that returns a value after it is executed.




Multithread:
  It is a technique that takes advantage of the current
high-code code creation through the simultaneous execution of several methods.




Memory Management
  It is an essential part in the process of optimizing a program, it is something that
should be taken into account from the very beginning of the program's creation. The simplest way of
memory is to manage the number of variables initialized by the program.
