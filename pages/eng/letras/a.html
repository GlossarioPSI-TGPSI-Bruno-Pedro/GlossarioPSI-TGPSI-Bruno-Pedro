Array: 
  An array is a data structure that stores a collection of elements of the same type.


  
Exemple:
  public class NewBehaviourScript : MonoBehaviour {

      int i;
      int x = new int[10];

  	void Start () {

          for (i = 0;i < 5;i++) {

              x[i] = i;

          }

  	}

  }

  
  To declare an array in C # it is necessary to create a variable of the type that we want
our array is, then we match this variable to "new type [number of spaces]"
in the case of the example above "new int [10]" this will transform the variable x into an array of types
with 10 spaces. To fill the array, use a for loop that will run through all
the positions in the array beginning in the case of the above example by position 0 and filling the array
to position 9.
  


Array Lists: 
  Array lists are a more modern version of normal arrays, in which the
programmer does not need to specify the size of the array but the size is dynamic,
changing according to the needs of the program.
