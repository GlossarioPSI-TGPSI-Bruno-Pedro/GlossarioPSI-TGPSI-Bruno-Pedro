Class:
  It is a set of functions. As for example, the bicycle class, it has as functions the
wheel, seat, etc.




Abstract Classes:
  They are models for other classes to inherit.




Sealed Classes:
  They are classes that allow to avoid the inheritance for other classes. Instead
of the abstract classes they can say who should or should not inherit them.




Compiler:
  It translates a program of a textual language easily understood by a being
for a "computer language" specific to a processor and operating system.




Contructor:
  It is a method called as soon as a new instance is created.




Classe Diagram:
  It is a representation of the structure and relationships of the classes that serve
from model to object.
