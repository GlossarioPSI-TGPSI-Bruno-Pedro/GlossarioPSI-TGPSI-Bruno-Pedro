Operator:
  It is a program element applied to one or more operands. There are unary operators
(++, -), arithmetic operators (+, -, *, /), conditional operators (&&, ||) and
relational operators (<,>, ==,! =).




Overloading:
  Is the ability to create multiple methods with the same name but with different
utilities.




Overriding:
  Is the ability to define a specific behavior for a subclass type, the
which means that a subclass can implement a parent class method based on its
requirement.
