Destructor:
  It is a member of a class, a method, invoked when the class must be destroyed.
It is normally used to free memory.



Exemple:
  class Car
  {
      ~Car()  // destructor
      {
          // cleanup statements...
      }
  }


}




Data types:
  Specify the memory space reserved for a variable and what the variable
will store. Example: int, float, char, etc.
