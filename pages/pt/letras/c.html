Inicio 
Classe:
  É um conjunto de funções. Como por exemplo, a classe bicicleta, ela tem como funções o
giador, as rodas, o assento, etc.




Classes abstratas:
  São modelos para outras classes poderem herdar.




Classes Seladas:
  São classes que permitem evitar a herança para outras classes. Ao contrário
das classes abstratas estas podem dizer quem deve ou não deve herda-las.




Compilador:
  Traduz um programa de uma linguagem textual facilmente entendida por um ser
humano para uma "linguagem de computador", específica para um processador e sistema operativo.




Contrutor:
  É um método chamado assim  que uma nova instancia for criada.
