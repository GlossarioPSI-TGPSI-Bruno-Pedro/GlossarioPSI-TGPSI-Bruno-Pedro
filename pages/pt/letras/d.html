Destrutor:
  É um membro de uma classe, um método, invocado quando a classe deve ser destruída.
É utilizado, normalmente, para libertar memória.



Exemplo:
  class Car
  {
      ~Car()  // destructor
      {
          // cleanup statements...
      }
  }


}




Diagrama de Classes:
  É uma representação da estrutura e relações das classes que servem
de modelo para objetos.
