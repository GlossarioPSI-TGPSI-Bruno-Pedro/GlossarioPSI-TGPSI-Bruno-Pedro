Array: 
  Um array é uma estrutura de dados que armazena uma coleção de elementos de tipos iguais.


  
Exemplo:
  public class NewBehaviourScript : MonoBehaviour {

      int i;
      int x = new int[10];

  	void Start () {

          for (i = 0;i < 5;i++) {

              x[i] = i;

          }

  	}

  }

  
  Para declarar um array em C# é necessário criar uma variavel do tipo que pretendemos que
o nosso array seja, em seguida igualamos essa variavel a "new tipo[numero de espaços]"
no caso do exemplo acima "new int[10]" isto vai transformar a variavel x num array de tipos
inteiro com 10 espaços. Para preencher o array utiliza-se um ciclo for que vai percorrer todas
as posições no array começando no caso do exemplo acima pela posição 0 e preenchendo o array
até à posição 9.
  


Array Lists: 
  Array lists são uma versão mais moderna dos arrays normais, nos quais o
programador não precisa de especificar o tamanho do array mas o tamanho é dinâmico,
alterando-se de acordo com as necessidades do programa.
